$F1 = \frac{2 \times precision \times recall}{precision + recall}$

$Macro\ average\ F1 = \frac{1}{K}\sum_{k=1}^{K} F1\ for\ class\ k$

# 竞赛背景

随着自动化运维和云计算环境的广泛应用，PowerShell 已成为 Windows 系统管理与运维的重要工具。然而，攻击者也越来越多地滥用 PowerShell 的强大功能来实施攻击，例如通过 下载并执行恶意代码、权限提升、横向移动以及数据窃取 等方式渗透目标系统。由于 PowerShell 脚本通常以文本形式存在，并且可以通过混淆、编码、动态执行等技术隐藏真实行为，传统基于特征码或简单规则的检测方法往往难以及时发现这些恶意脚本。因此，如何在海量脚本数据中自动识别潜在的恶意 PowerShell 脚本，已成为主机安全与威胁检测领域的重要研究问题之一。

本竞赛聚焦于 PowerShell 恶意脚本检测 任务，参赛者需要基于给定的 PowerShell 脚本数据，结合 机器学习、深度学习、自然语言处理等技术，构建高效的分类模型，对脚本进行自动化分析并判断其是否具有恶意行为。竞赛数据集中包含 正常运维脚本与恶意攻击脚本的相关特征，其中恶意样本可能包含代码混淆、编码执行、动态调用等复杂特征，同时数据也具有 脚本结构差异大、行为模式隐蔽、特征分布不均衡 等特点，从而模拟真实环境中恶意脚本检测的实际难度。

竞赛旨在推动 智能化主机安全检测技术 的发展，鼓励参赛者探索更加高效和鲁棒的恶意脚本识别方法，提高对 PowerShell 恶意行为的自动化检测能力，并促进安全研究社区对脚本型攻击技术的理解与防御能力。竞赛通过提供相关数据集，使参赛者能够构建模型来识别 PowerShell 脚本中的潜在恶意行为模式。
