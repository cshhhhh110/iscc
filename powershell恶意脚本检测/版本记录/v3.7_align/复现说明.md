# v3.7_align 复现说明

## 关键改动

exact_key5 特征对齐：学生模型输入从 132 维扩展到 137 维，新增 5 个 exact_key 频次特征。

这 5 个特征来自 teacher 的特征工程（`reproduce_teacher_npz.py` 的 `FeatureRecipe`）：
- `exact_key_freq_train` — train 侧 key 频次
- `exact_key_freq_test` — test 侧 key 频次
- `exact_key_freq_all` — train+test 合并频次
- `exact_key_logfreq_train` — train 侧 log1p 频次
- `exact_key_logfreq_test` — test 侧 log1p 频次

## 根因

teacher/student 在训练集内的表示已经足够接近，真正缺的是 test 端 key 频次结构信息。这 5 个特征提供了多视角的 exact_key 分布信号，属于合法的 transductive feature（不使用 test label）。

## 效果

- class1 从 3308 提升到 3354 (+46)
- c2 保持在 3553（甜区 3450-3650）
- 平台 Macro-F1: **0.70831**（v3.3: 0.70441，+0.0039）
