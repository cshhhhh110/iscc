## 三条成功经验

### 1. 新特征 > 新架构 > 训练策略
- 反汇编特征（v2.3/v2.4）：+0.014 平台提升，最大单次跃迁
- 训练策略（v1.9 Cosine+SWA）：+0.010
- 架构改动（v2.0 self-attention）：-0.005，过拟合
- **结论**：特征信息量决定天花板，架构和策略只是逼近它

### 2. 数据质量 > 数据数量
- v1.5：17K 伪标签（0.9阈值）→ 0.923
- v1.6：28K 伪标签（0.7阈值）→ 0.904
- 低置信度样本的CWE标注在86类任务上错误率极高

### 3. 本地虚高 = 平台必跌
- v1.6 本地 0.925 → 平台 0.904（伪标签过拟合）
- v2.0 本地 0.907 → 平台 0.916（增强过拟合）
- v2.2 本地 0.913 → 平台 0.910（训练策略过拟合）
- 健康的本地范围：0.895-0.920

## 三条失败教训

### 1. 逐CWE类阈值
在训练集val上拟合86个阈值，搬到测试集后过度降级1387个label=1预测（v1.7: 0.876）

### 2. Self-attention in byte encoder
连续两个版本拖累平台（v2.0/v2.1），更多参数在有限数据上过拟合

### 3. 稀有CWE增强
字节翻转生成非法指令序列，模型学到噪声模式而非真实漏洞特征

## 技术架构（最终版 v2.4）

```
                    ┌──────────────────────────┐
  binary file ──────▶│ 446-dim PE元数据特征      │──▶ LightGBM ──┐
                    │ + 192-dim Capstone反汇编  │   label+CWE    │
                    │ + 36-dim 扩展反汇编       │                │
                    │ = 674-dim tabular         │                ├── scalar fusion ──▶ 预测
                    └──────────────────────────┘                │
                                                                 │
                    ┌──────────────────────────┐                │
  binary file ──────▶│ 8192-byte raw窗口        │──▶ ByteCNN ───┘
                    │ Embedding(48) → 5×Conv   │   label+CWE双头
                    │ → avg+max pool → MLP     │
                    └──────────────────────────┘

  训练: CosineWarmRestarts + 3-epoch warmup + SWA + 30 epochs
  推理: 3-seed加权集成 + scalar fusion + TTA(3窗口)
  伪标签: 17,343条（v1.5模型, 阈值0.9）
```

## 反汇编特征明细

| 类别 | 维数 | 内容 |
|------|------|------|
| 指令统计 | 5 | 总指令数、唯一opcode数、熵、分支密度、访存密度 |
| Opcode unigram | ~150 | 常见x86指令频率 |
| Opcode bigram | ~35 | push|call, cmp|je 等关键指令对 |
| 寄存器热度 | 26 | eax~rip各寄存器引用频率 |
| 函数边界 | 3 | 函数数量、平均/最大指令数 |
| 操作数模式 | 3 | 栈操作比、立即数比、内存引用比 |
| 指令长度 | 4 | 平均/标准差、长指令比(>4B)、超长指令比(>6B) |

## 关键设计决策

- **scalar fusion 而非 MLP fusion**：MLP在86类任务上过拟合严重（held-out 0.72 vs scalar 0.90）
- **3-seed 而非 4-seed**：第4个种子增加方差，抵消集成收益
- **种子 [42, 123, 202]**：456 在6个版本中从未超过0.873，202替代后方差从0.084降至0.028
- **伪标签阈值 0.9**：0.7的实验证明低阈值反噬
- **不做数据增强**：字节翻转产生非法指令
