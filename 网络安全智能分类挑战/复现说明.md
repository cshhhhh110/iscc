# 复现说明 — submission_blend_top2_ft (平台 0.709)

## 原理

训练两组 FT-Transformer 并对测试集概率做 50:50 平均：

| 组 | Seeds | Folds | 模型数 | 模拟版本 | 平台得分 |
|----|-------|-------|--------|---------|---------|
| Group A | [20260504] | 5 | 5 | v1.1 | 0.708 |
| Group B | [20260504, 20260505] | 5 | 10 | v1.4 | 0.707 |
| **融合** | — | — | **15** | — | **0.709** |

## 环境

```
Python 3.10
PyTorch 2.x + CUDA (GPU 推荐，CPU 也可运行)
pandas, numpy, scikit-learn, tqdm
```

环境安装：
```powershell
conda create -n iscc-gpu python=3.10 -y
conda activate iscc-gpu
pip install torch pandas numpy scikit-learn tqdm
```

## 模型架构

FT-Transformer: 50 维输入 → Tokenizer(d=64) → 4×TransformerBlock(h=8, ffn=192) → Head(12类输出)

预处理: median + IQR 标准化，每折独立拟合

超参数: lr=8e-4, wd=2e-4, label_smoothing=0.03, epochs=90, patience=14, batch=1024
